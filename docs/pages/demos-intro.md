# Demos

Live WASM demos running directly in your browser  -  no install required. Each demo showcases a different aspect of the Willow framework.

## Categories

**Stress Tests**  -  Performance benchmarks pushing thousands of sprites simultaneously.

**Interactive**  -  Hands-on demos with physics, puzzles, animations, filters, and more.

Use the sidebar to browse individual demos, or launch them directly in a new tab.
