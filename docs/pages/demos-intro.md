# Demos

Live WASM demos running directly in your browser  -  no install required. Each demo showcases a different aspect of the Willow framework.

## Categories

**Stress Tests**  -  Push the rendering pipeline with thousands of sprites.

**Interactive**  -  Physics, puzzles, tweens, filters, lighting, and underwater scenes you can click and play with.

## Run locally

Clone the repo and use `go run`:

```bash
git clone https://github.com/phanxgames/willow.git
cd willow
go run ./demos/<name>
```

Browse the [demos/ directory on GitHub](https://github.com/phanxgames/willow/tree/main/demos) for full source.

Use the sidebar to browse individual demos.
